# python

A home for my Python projects, experiments. 

## What's in here

### 📒 `00_python_cheat_sheet.md`
A pandas cheat sheet for people who don't write Python every day. Organised by **what you're trying to do** ("filter rows",
"count unique things") rather than by function name — because nobody remembers
the function name.

Built for quick lookup: `Ctrl+F` the thing you need, grab the snippet, leave.

More will land here over time.

## Who this is for

Anyone who:
- codes in bursts and loses the syntax in between
- wants the *useful* 20% of pandas, not the whole API
- likes a reference they can actually skim

## Using it

Just read the files straight here on GitHub. Want your own copy to scribble on?
Hit **Fork** (top right) and make it yours.
